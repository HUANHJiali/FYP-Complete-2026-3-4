## 4. 问题诊断

### 4.1 登录问题根因分析

**症状**:
- 管理员（type=0）可以登录
- 教师（type=1）和学生（type=2）无法登录
- 数据库查询正常
- 输入验证器正常
- Python shell模拟正常

**可能原因**:

1. **输入验证问题** ⭐⭐⭐
   - `InputValidator.validate_username()` 可能在HTTP请求中行为不同
   - 需要检查HTTP请求的实际数据格式

2. **中间件拦截** ⭐⭐
   - 可能有中间件在过滤特定用户类型
   - 需要检查MIDDLEWARE配置

3. **字符编码问题** ⭐
   - HTTP请求中的字符编码可能有问题
   - 需要检查请求解析逻辑

4. **权限检查问题** ⭐⭐⭐
   - 可能在登录前就进行了某种权限过滤
   - 需要检查URL路由配置

### 4.2 权限问题分析

**症状**: 多个API返回"权限非法"

**可能原因**:
1. 权限装饰器逻辑错误
2. 角色检查过于严格
3. 可能缺少必要的认证头
