## 🔧 后端实现

### 1. OperationLogger 工具类

**文件位置**: `source/server/utils/OperationLogger.py`

**核心方法**:
```python
# 通用日志记录
OperationLogger.log(user_id, user_name, user_type, operation_type, module_name, ...)

# 登录日志
OperationLogger.log_login(user_id, user_name, user_type, status, request, error_message)

# 登出日志
OperationLogger.log_logout(user_id, user_name, user_type, request)

# CRUD 操作日志
OperationLogger.log_create(...)
OperationLogger.log_update(...)
OperationLogger.log_delete(...)
OperationLogger.log_submit(...)
```

**使用示例**:
```python
from utils.OperationLogger import OperationLogger

# 记录登录成功
OperationLogger.log_login(
    user_id=user.id,
    user_name=user.name,
    user_type=user.type,
    status=OperationLogger.STATUS_SUCCESS,
    request=request
)

# 记录创建考试
OperationLogger.log_create(
    user_id=request.user.id,
    user_name=request.user.name,
    user_type=request.user.type,
    module_name=OperationLogger.MODULE_EXAM,
    resource_id=exam.id,
    resource_name=exam.name,
    operation_detail={'duration': 60, 'question_count': 10},
    request=request

```

### 2. LogViews API 视图

**文件位置**: `source/server/app/views/log_views.py`

**支持的接口**:

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/logs/` | 查询操作日志（支持分页和过滤） |
| DELETE | `/api/logs/` | 批量删除操作日志 |
| PUT | `/api/logs/` | 导出操作日志为 CSV |

**查询参数** (GET):
```
pageIndex: 页码 (默认 1)
pageSize: 每页数量 (默认 10)
userId: 用户ID (可选)
userName: 用户姓名 (可选)
operationType: 操作类型 (可选)
moduleName: 模块名称 (可选)
status: 状态 1/0 (可选)
startDate: 开始日期 YYYY-MM-DD (可选)
endDate: 结束日期 YYYY-MM-DD (可选)
```

**示例请求**:
```bash
# 查询所有登录日志
GET /api/logs/?operationType=login&pageSize=20

# 查询某用户的失败操作
GET /api/logs/?userId=123&status=0

# 导出CSV
PUT /api/logs/
Body: {"filters": {"startDate": "2026-01-01", "endDate": "2026-12-31"}}
```

## 📝 使用指南

### 1. 记录日志

在任何需要记录日志的地方，使用 OperationLogger：

```python
# 在 views 中导入
from utils.OperationLogger import OperationLogger

# 记录操作
OperationLogger.log_create(
    user_id=current_user.id,
    user_name=current_user.name,
    user_type=current_user.type,
    module_name=OperationLogger.MODULE_EXAM,
    resource_id=new_exam.id,
    resource_name=new_exam.name,
    request=request
)
```

### 2. 查看日志

**方式一：通过前端页面**
1. 访问 `http://localhost:8080/#/home/adminLogs`
2. 使用筛选条件查询日志
3. 点击"详情"查看完整信息

**方式二：通过 API**
```bash
curl "http://localhost:8000/api/logs/?pageIndex=1&pageSize=20&operationType=login"
```

### 3. 导出日志

在前端页面点击"导出日志"按钮，或通过 API:
```bash
curl -X PUT "http://localhost:8000/api/logs/" \
  -H "Content-Type: application/json" \
  -d '{"filters": {"startDate": "2026-01-01", "endDate": "2026-12-31"}}' \
  --output logs.csv
```
