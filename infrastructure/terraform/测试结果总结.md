## 🚀 如何使用

### 快速开始

1. **配置变量**
   ```bash
   cd infrastructure/terraform
   cp terraform.tfvars.example terraform.tfvars
   # 编辑 terraform.tfvars
   ```

2. **配置 AWS 凭证**
   ```bash
   export AWS_ACCESS_KEY_ID="your-key"
   export AWS_SECRET_ACCESS_KEY="your-secret"
   export AWS_SESSION_TOKEN="your-token"  # Learner Lab
   ```

3. **部署**
   ```bash
   terraform init
   terraform plan
   terraform apply
   ```
