# 前端登录问题诊断指南

## 问题：前端无法登录

## 请帮我检查以下内容：

### 1. 浏览器控制台错误

请按F12打开开发者工具，然后：

**Console标签** - 查看是否有错误：
- 截图所有红色错误信息
- 复制错误文本

**Network标签** - 查看登录请求：
1. 刷新页面
2. 尝试登录
3. 找到 `login` 或 `/api/login/` 请求
4. 点击查看详情：
   - **Request URL**: 是什么？
   - **Status Code**: 是多少？（200, 404, 500?）
   - **Request Headers**: 特别是 Content-Type
   - **Request Payload** 或 **Request Data**: 发送的数据
   - **Response**: 返回的内容

### 2. 确认访问地址

请确认你访问的地址是：
- ✅ http://localhost:8080
- ❌ 不要是 http://127.0.0.1:8080

### 3. 测试账号

请使用的账号：
```
用户名: admin
密码: 123456
```

### 4. 登录步骤

1. 打开 http://localhost:8080
2. 应该看到登录页面
3. 输入用户名: admin
4. 输入密码: 123456
5. 点击"登录系统"按钮

### 5. 出现什么问题？

请告诉我：
- [ ] 页面无法打开
- [ ] 页面显示但登录按钮没反应
- [ ] 点击登录后显示错误提示（是什么错误？）
- [ ] 点击登录后没有任何反应
- [ ] 登录后显示"网络错误"
- [ ] 其他（请描述）

### 6. 浏览器信息

- 浏览器名称（Chrome/Edge/Firefox?）
- 版本号
