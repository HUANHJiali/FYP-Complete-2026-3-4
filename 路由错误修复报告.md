# 练习结果页面路由错误修复报告

**修复时间**: 2026-02-08 18:20
**错误**: 点击练习试卷"查看结果"按钮时出现路由错误

---

## 🔍 错误信息

```
ERROR
No match for
 {"name":"practiceResult","query":{"paperId":1,"logId":null},"params":{}}
```

**错误位置**: `practises.vue` 第204行，`onOk` 回调函数

**错误���因**: 当用户点击练习试卷的"查看结果"按钮时，系统尝试跳转到 `practiceResult` 页面，但传递的 `logId` 参数为 `null`，导致路由无法匹配。

---

## 🔎 问题分析

### 1. 按钮显示逻辑问题

**原代码** (practises.vue 第161-177行):
```vue
<Button
    v-else-if="paper.status === 'in_progress'"
    type="warning"
    @click="continuePractice(paper)">
    继续练习
</Button>
<Button
    v-else
    type="success"
    @click="reviewPaper(paper)">
    查看结果
</Button>
```

**问题**:
- 当 `paper.status !== 'in_progress'` 时，显示"查看结果"按钮
- 但没有检查 `paper.logId` 是否存在
- 如果用户还没有完成练习，`logId` 为 `null`

### 2. 跳转逻辑问题

**原代码** (practises.vue 第788-797行):
```javascript
onOk: () => {
    // 跳转到结果详情页面
    this.$router.push({
        name: 'practiceResult',
        query: {
            paperId: paper.id,
            logId: paper.logId || null  // ❌ logId为null时仍然跳转
        }
    });
}
```

**问题**:
- 即使 `logId` 为 `null`，仍然尝试跳转
- `practiceResult.vue` 页面要求必须有 `logId`
- 导致路由匹配失败

### 3. practiceResult.vue 验证逻辑

**代码** (practiceResult.vue 第502-505行):
```javascript
if (!this.logId) {
    this.$Message.error('缺少练习记录ID');
    return;
}
```

- 页面要求必须有 `logId` 参数
- 如果 `logId` 为空，显示错误但不跳转
- 这与路由行为不一致

---

## ✅ 修复方案

### 1. 修改按钮显示条件

**修改后**:
```vue
<Button
    v-else-if="paper.status === 'in_progress'"
    type="warning"
    @click="continuePractice(paper)">
    继续练习
</Button>
<Button
    v-else-if="paper.logId"
    type="success"
    @click="reviewPaper(paper)">
    查看结果
</Button>
```

**改进**:
- 只有在有 `logId` 时才显示"查看结果"按钮
- 没有练习记录时不显示该按钮
- 避免用户点击无效按钮

### 2. 修改跳转逻辑

**修改后**:
```javascript
onOk: () => {
    // 只有在有logId时才跳转到结果详情页面
    if (paper.logId) {
        this.$router.push({
            name: 'practiceResult',
            query: {
                paperId: paper.id,
                logId: paper.logId
            }
        });
    } else {
        this.$Message.warning('暂无练习记录，请先完成练习');
    }
}
```

**改进**:
- 添加 `logId` 存在性检查
- 只有在有练习记录时才跳转
- 没有记录时显示友好提示
