## 调试步骤（如果登录失败）

### 查看浏览器控制台
1. 按F12打开开发者工具
2. 切换到**Console**标签
3. 尝试登录
4. 查看是否有红色错误信息
5. 截图或复制错误信息

### 查看网络请求
1. 在开发者工具中
2. 切换到**Network**标签
3. 尝试登录
4. 找到**login**请求
5. 点击查看详细信息：
   - Request URL
   - Request Headers
   - Request Payload（发送的数据）
   - Response（返回的数据）
