## 📝 使用示例

### 查询不同类型的日志

```bash
# 查询所有登录日志
curl "http://localhost:8000/api/logs/?operationType=login"

# 查询失败的登录
curl "http://localhost:8000/api/logs/?operationType=login&status=0"

# 查询某个用户的日志
curl "http://localhost:8000/api/logs/?userId=1"

# 按日期范围查询
curl "http://localhost:8000/api/logs/?startDate=2026-02-01&endDate=2026-02-28"
```

### 在代码中记录日志

```python
from utils.OperationLogger import OperationLogger

# 记录创建操作
OperationLogger.log_create(
    user_id=request.user.id,
    user_name=request.user.name,
    user_type=request.user.type,
    module_name=OperationLogger.MODULE_EXAM,
    resource_id=exam.id,
    resource_name=exam.name,
    request=request
)
```
