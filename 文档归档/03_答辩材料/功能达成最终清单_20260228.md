## 4. 本轮已修复项（2026-02-28）

1) 学生雷达图 API 断链修复
- 现象：`RadarChart` 调用 `getStudentExams`，但 API 层缺少导出。
- 修复：在前端 API 层新增 `getStudentExams` 并复用 `examlogs/pagestu`。
- 结果：雷达图数据链路闭环。

2) 管理员仪表盘快捷入口路由修复
- 现象：`adminDashboard` 中使用旧 `/admin/*` 跳转，存在不可达风险。
- 修复：新增路径到路由名映射（`adminUsers/adminSubjects/adminExams/...`）。
- 结果：快捷入口稳定可达。

3) 高级分析页入口打通
- 新增主路由：`dataDashboard`、`classComparison`、`studentProgress`
- 新增管理员菜单：数据大屏、班级对比分析、学生进步分析
- 新增仪表盘快捷入口：上述三项分析页
