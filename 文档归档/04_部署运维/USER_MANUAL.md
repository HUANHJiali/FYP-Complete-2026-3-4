## 1. 系统概述

### 1.1 系统简介

智能在线考试系统是一个功能完善的在线教育平台，支持：
- ✅ 在线考试与练习
- ✅ AI智能评分
- ✅ 错题本管理
- ✅ 学习数据分析
- ✅ 智能题目推荐

### 1.2 用户角色

| 角色 | 权限 |
|------|------|
| **管理员** | 系统管理、用户管理、数据统计 |
| **教师** | 题库管理、考试管理、成绩分析 |
| **学生** | 参加考试、在线练习、查看成绩 |

### 1.3 系统访问

- **前端地址**: http://localhost:8080
- **后端API**: http://localhost:8000/api/

## 2. 快速开始

### 2.1 登录系统

1. 打开浏览器，访问系统首页
2. 输入用户名和密码
3. 点击"登录"按钮

**默认测试账号**:

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 管理员 | admin | 123456 |
| 教师 | teacher | 123456 |
| 学生 | student | 123456 |

### 2.2 修改密码

1. 登录后点击右上角头像
2. 选择"个人设置"
3. 点击"修改密码"
4. 输入旧密码和新密码
5. 点击"保存"

## 3. 管理员指南

### 3.1 仪表盘

登录后进入管理员仪表盘，可查看：
- 学生总数
- 教师总数
- 考试总数
- 题目总数
- 最近活动

### 3.2 学院管理

**添加学院**:
1. 点击左侧菜单"学院管理"
2. 点击"添加学院"按钮
3. 输入学院名称
4. 点击"保存"

**编辑学院**:
1. 在学院列表中找到目标学院
2. 点击"编辑"按钮
3. 修改学院信息
4. 点击"保存"

### 3.3 班级管理

**添加班级**:
1. 点击"班级管理"
2. 点击"添加班级"
3. 输入班级名称
4. 选择所属学院
5. 点击"保存"

### 3.4 学生管理

**添加学生**:
1. 点击"学生管理"
2. 点击"添加学生"
3. 填写学生信息
4. 点击"保存"

**批量导入学生**:
1. 点击"批量导入"按钮
2. 下载CSV模板
3. 按模板填写学生信息
4. 上传CSV文件
5. 系统自动导入

### 3.5 教师管理

操作流程与学生管理类似。

### 3.6 数据统计

**查看统计报表**:
1. 点击"数据统计"
2. 选择统计类型：
   - 考试统计
   - 学生统计
   - 班级统计
   - 科目统计
3. 查看图表和数据

## 4. 教师指南

### 4.1 题库管理

**添加题目**:
1. 点击"题库管理"
2. 点击"添加题目"
3. 选择题目类型：
   - 选择题
   - 填空题
   - 判断题
   - 简答题
   - 编程题
   - 综合题
4. 填写题目内容、答案、解析
5. 点击"保存"

**AI生成题目**:
1. 点击"AI生成题目"
2. 输入科目和主题
3. 选择难度等级
4. 选择题目类型
5. 设置生成数量
6. 点击"生成"
7. 预览并保存需要的题目

### 4.2 考试管理

**创建考试**:
1. 点击"考试管理"
2. 点击"创建考试"
3. 填写考试信息：
   - 考试名称
   - 考试时间
   - 考试时长
   - 参与班级
4. 选择题目或使用AI组卷
5. 点击"发布"

**监考管理**:
1. 在考试进行时，点击"监考"
2. 查看学生答题进度
3. 查看异常行为记录

**成绩批改**:
1. 考试结束后，点击"批改"
2. 查看学生答案
3. AI自动评分
4. 手动调整分数
5. 添加评语
6. 发布成绩

### 4.3 练习管理

**创建练习试卷**:
1. 点击"练习管理"
2. 点击"创建试卷"
3. 设置试卷信息
4. 选择题目
5. 发布给学生

### 4.4 成绩分析

**查看班级成绩**:
1. 点击"成绩分析"
2. 选择班级和考试
3. 查看统计数据：
   - 平均分
   - 最高分/最低分
   - 及格率
   - 分数分布

**班级对比**:
1. 点击"班级对比"
2. 选择多个班级
3. 查看对比图表

## 5. 学生指南

### 5.1 考试中心

**参加考试**:
1. 点击"考试中心"
2. 查看待考试列表
3. 点击"进入考试"
4. 阅读考试须知
5. 开始答题
6. 提交试卷

**答题技巧**:
- 选择题：点击选项即可选择
- 填空题：在输入框中填写答案
- 判断题：选择正确或错误
- 编程题：在代码编辑器中编写代码

### 5.2 练习中心

**开始练习**:
1. 点击"练习中心"
2. 选择练习试卷
3. 点击"开始练习"
4. 答题
5. 查看即时反馈

**查看解析**:
- 每道题答完后可查看解析
- 了解答题思路

### 5.3 错题本

**查看错题**:
1. 点击"错题本"
2. 查看所有错题
3. 按科目筛选

**复习错题**:
1. 点击错题进入详情
2. 查看正确答案和解析
3. 点击"标记已复习"
4. 系统会定期提醒复习

**导出错题**:
1. 点击"导出CSV"
2. 下载错题文件
3. 可打印或离线复习

### 5.4 成绩查询

**查看成绩**:
1. 点击"我的成绩"
2. 查看考试成绩列表
3. 点击详情查看答题情况

**进步曲线**:
1. 点击"进步分析"
2. 选择时间范围
3. 查看成绩趋势图
4. 了解学习进步情况

**雷达图分析**:
1. 查看各科目得分情况
2. 发现薄弱科目
3. 获取改进建议

### 5.5 任务中心

**查看任务**:
1. 点击"任务中心"
2. 查看待完成任务
3. 点击任务进入详情

**提交任务**:
1. 完成任务要求
2. 上传附件（如有）
3. 点击"提交"
4. 等待教师批改

### 5.6 个人中心

**修改个人信息**:
1. 点击右上角头像
2. 选择"个人中心"
3. 修改个人信息
4. 点击"保存"

**主题设置**:
1. 点击"主题设置"
2. 选择浅色/深色模式
3. 选择主题色
4. 设置字体大小

**消息中心**:
1. 点击"消息中心"
2. 查看系统通知
3. 标记已读

## 6. 常见问题

### 6.1 登录问题

**Q: 忘记密码怎么办？**

A: 联系管理员重置密码，或使用默认密码登录后修改。

**Q: 账号被锁定怎么办？**

A: 连续多次密码错误会导致锁定，联系管理员解锁。

### 6.2 考试问题

**Q: 考试中网络断开怎么办？**

A: 系统会自动保存答案，重新连接后可继续答题。

**Q: 超时未提交会怎样？**

A: 系统会自动提交当前答案。

**Q: 如何查看考试答案？**

A: 考试结束后，教师开放答案后可查看。

### 6.3 练习问题

**Q: 练习可以重复做吗？**

A: 可以，每次练习都会记录成绩。

**Q: 练习成绩会记入总成绩吗？**

A: 练习成绩不计入考试总分，但会作为学习参考。

### 6.4 错题本问题

**Q: 错题如何自动收集？**

A: 考试和练习中答错的题目会自动添加到错题本。

**Q: 如何删除错题？**

A: 点击错题详情，选择"移除"。

### 6.5 技术问题

**Q: 支持哪些浏览器？**

A: 推荐使用Chrome、Firefox、Edge最新版本。

**Q: 手机可以访问吗？**

A: 支持，系统已适配移动端。

**Q: 文件上传限制？**

A: 单个文件最大50MB，支持PDF、DOC、图片等格式。
