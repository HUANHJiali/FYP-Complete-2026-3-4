### 优化1: 修复定时器内存泄漏 ✅

**文件**: `source/client/src/views/pages/answer.vue`

**问题描述**:
- `setInterval`定时器在组件销毁时未清理
- 可能导致内存泄漏，特别是在频繁切换页面时

**修复内容**:
```javascript
// 1. 在data中添加timer引用
data() {
  return {
    timer: null  // 定时器引用，用于清理
  }
}

// 2. 保存定时器引用
this.timer = setInterval(() => {...}, 1000)

// 3. 在beforeUnmount中清理
beforeUnmount() {
  if (this.timer) {
    clearInterval(this.timer)
    this.timer = null
  }
}
```

**影响**: 🔴 高 - 防止内存泄漏，提升长期稳定性

**测试**: ✅ 验证通过

### 优化5: 创建Token获取混入 ✅

**文件**: `source/client/src/mixins/authMixin.js`

**问题描述**:
- 13个组件中重复相同的token获取逻辑
- 代码重复，维护困难

**修复内容**:
```javascript
// authMixin.js
export default {
  computed: {
    token() {
      return this.$store.state.token || sessionStorage.getItem('token') || ''
    },
    isLoggedIn() {
      return !!this.token
    }
  },

  methods: {
    checkLogin(message = '请先登录') {
      if (!this.isLoggedIn) {
        this.$Message.error(message)
        this.$router.push('/login')
        return false
      }
      return true
    },
    // ... 更多方法
  }
}
```

**使用示例**:
```javascript
// 替换前
const token = this.$store.state.token || sessionStorage.getItem('token')
getLoginUser(token).then(...)

// 替换后
import authMixin from '@/mixins/authMixin'

export default {
  mixins: [authMixin],

  methods: {
    loadData() {
      getLoginUser(this.token).then(...)  // 简洁！
    }
  }
}
```

**影响**: 🟡 中 - 提升代码复用性

**收益**:
- 减少约80行重复代码
- 统一认证逻辑
- 便于维护
