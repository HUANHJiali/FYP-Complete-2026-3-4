# 答辩口播稿（1分钟最终版）

各位老师好，我的 FYP 是一个基于 Django + Vue 的智能考试系统，覆盖管理员、教师、学生三类角色，核心功能包括考试管理、练习系统和 AI 评分。

这次我重点完成了三件事。
第一，做了最小安全基线，不做过度工程：统一了配置策略，完成了 CSRF 白名单豁免、限流开关和环境变量化。
第二，建立了可复跑回归基线，覆盖登录、考试、练习、AI评分四条主链路，并做成一键脚本，能稳定复现。
第三，做了低风险性能优化，对高频分页接口加了短缓存，提升列表场景稳定性。

我的目标不是只“把功能做出来”，而是把系统做成“可演示、可验证、可交付”。
现场我可以直接运行一键校验脚本，展示系统检查、回归通过和健康状态。谢谢老师。
