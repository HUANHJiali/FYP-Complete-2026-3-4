## ⚠️ 发现的问题

### 1. AI 服务配置问题（低优先级）

**问题描述**:
```json
"ai_service": "unhealthy: attempted relative import with no known parent package"
```

**影响**: AI 智能评分和自动出题功能可能不可用
**优先级**: 🟡 中优先级
**建议**: 检查 AI 配置和导入路径

---

### 2. 错题本路由问题（低优先级）

**问题描述**: `/api/wrongquestions/page/` 返回 404

**影响**: 错题本功能可能无法访问
**优先级**: 🟡 中优先级
**建议**: 检查 URL 路由配置

---

### 3. URL 斜杠要求（已说明）

**问题描述**: POST 请求需要在 URL 末尾加 `/`

**示例**:
```bash
# ❌ 错误
POST /api/login

# ✅ 正确
POST /api/login/
```

**影响**: 前端 API 调用需要统一加斜杠
**优先级**: 🟢 低优先级（前端已配置好）
