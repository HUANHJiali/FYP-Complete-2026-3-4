# 学生上云快速开始（5步）

## Step 1: 上传脚本到云服务器
把以下文件放到服务器任意目录（推荐 home 目录）：
- 13_setup_server_student.sh
- 14_env_prod_student_template.env

## Step 2: 执行一键部署脚本
先将命令中的占位符替换为真实值（仓库地址、域名或公网 IP）：
```bash
chmod +x 13_setup_server_student.sh
bash 13_setup_server_student.sh --repo-url <你的仓库地址> --domain <你的域名或公网IP> --app-dir /opt/fyp
```

## Step 3: 编辑项目根目录 .env.prod
脚本首次执行会创建 `.env.prod`，你必须修改这些字段：
- SECRET_KEY
- MYSQL_ROOT_PASSWORD / DB_PASSWORD
- ALLOWED_HOSTS
- CORS_ALLOWED_ORIGINS
- CSRF_TRUSTED_ORIGINS
- ZHIPUAI_API_KEY（如需AI）

## Step 4: 二次启动应用
```bash
cd /opt/fyp
docker compose -f docker-compose.prod.yml --env-file .env.prod up -d --build
```

## Step 5: 验收
- 主页：`http://<domain_or_ip>`
- 健康检查：`http://<domain_or_ip>/api/health/`
- 容器状态：
```bash
cd /opt/fyp
docker compose -f docker-compose.prod.yml --env-file .env.prod ps
```

## 可选：启用 HTTPS
```bash
sudo apt install -y certbot python3-certbot-nginx
sudo certbot --nginx -d <你的域名>
```

## 出错时先看日志
```bash
cd /opt/fyp
docker compose -f docker-compose.prod.yml --env-file .env.prod logs backend --tail 200
docker compose -f docker-compose.prod.yml --env-file .env.prod logs frontend --tail 200
```
